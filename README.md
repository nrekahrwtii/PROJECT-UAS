# PetHouseDB

## Requirement
- PHP 8.x, XAMPP, MySQL/MariaDB

## Setup
1. Clone repo
2. Copy includes/db.example.php -> includes/db.php dan isi kredensial
3. Import db_init.sql via phpMyAdmin atau:
   mysql -u root -p < db_init.sql
4. Taruh project di C:\xampp\htdocs\pethousedb
5. Jalankan http://localhost/pethousedb/pages/login.php
